# AI生成原型设计【AI口语·作文教练】

## 任务概述

请设计一个[AI口语·作文教练]的高保真原型图，这是一个[英语口语练习与作文批改]应用。原型图应该真实展示用户界面和核心交互流程，能够直接指导开发。

## 内容规划

请按照以下步骤规划内容：

1. 用户需求分析：分析该应用的核心用户需求和使用场景
2. 信息架构设计：确定主要页面和信息流转关系
3. 交互流程设计：设计核心交互流程和交互方式
4. 界面实现：使用HTML+TailwindCSS实现所有界面

## 设计风格

- 整体风格：现代简约风格，符合[iOS]设计规范
- 字体选择：使用系统字体栈（-apple-system, PingFang SC），保证在移动设备上的可读性
- 视觉元素：使用简洁的emoji图标和适当的圆角卡片增强视觉表现
- 交互效果：添加自然的过渡动画和微交互效果
- 圆角风格：卡片16px圆角，按钮12px圆角，输入框24px圆角

## 技术要求

- 开发框架：使用Vue 3 + Vite构建前端界面
- UI组件库：使用Vant 4移动端组件库
- 状态管理：使用Pinia管理全局状态
- 后端框架：使用Python FastAPI构建API服务
- 数据库：使用SQLite + SQLAlchemy ORM
- AI服务：使用OpenAI兼容接口（支持DeepSeek等）
- 脚本要求：添加必要的JavaScript实现交互效果
- 响应式要求：优化为移动设备视图，最大宽度430px，模拟真实手机尺寸
- 性能考虑：确保页面加载迅速，动画流畅
- 认证方式：JWT Token认证，支持注册/登录

## 资源配置

- 图标库：使用Vant内置图标 + emoji图标
- 外部引用：通过npm引入所需资源
- 代码质量：结构清晰，有适当注释

## 交付标准

- 兼容性：确保在主流浏览器上正常显示（Chrome/Edge/Safari/Firefox）
- 文件结构：前后端分离架构
  - frontend/ — Vue 3前端项目
  - backend/ — FastAPI后端项目
- 前端入口：frontend/index.html
- 后端入口：backend/run.py

## 特殊要求

- 创建底部导航栏（TabBar）实现首页、口语、作文、学习四个页面切换
- 模拟真实手机外观，最大宽度430px居中显示
- 所有界面在同一个SPA应用中，通过Vue Router切换
- 产品功能覆盖：口语AI对话、作文AI批改、全真模考、专项提升、学习报告
- 数据真实：所有数据从后端API获取，支持用户注册登录后的个性化数据

## 页面清单

### 页面1：首页（Home）
- 用户问候头部（头像、昵称、今日学习数据）
- 月度目标环形进度条
- 近14天打卡记录点（已打卡蓝色、今日橙色、未打卡灰色）
- 本周数据统计（连续天数、练习次数、学习分钟）
- 快捷功能入口（口语练习、作文批改、全真模考、专项训练）+ 剩余次数标签
- 快速练习入口（雅思口语、议论文批改、发音纠正、错题复习）
- 今日推荐列表（基于用户弱项AI推荐）

### 页面2：口语陪练（Speak）
- 顶部标题栏 + 评分详情入口
- 可横向滚动的场景选择标签（自由对话、职场面试、雅思模考、旅行出行、商务谈判、校园生活）
- AI角色卡（小鱼头像、名称、当前场景、实时评分）
- 对话消息列表（AI白色气泡左对齐、用户蓝色气泡右对齐）
- 地道表达提示卡片（绿色背景）
- 实时评分卡（4维进度条：发音、语法、词汇、流利度）
- 底部输入栏（文本输入框 + 发送按钮 + 语音按钮）
- 评分详情弹窗（综合评分、4维进度条、标签、AI建议）

### 页面3：作文批改（Write）
- 题目选择（支持弹窗切换，显示考试类型和文体标签）
- 作文输入文本域（实时字数统计）
- 批改模式切换（快速批改/深度批改）
- 批改结果：SVG圆环评分（满分9分制）
- 四维评分卡片（语法、逻辑、词汇、结构 + 进度条）
- 总体评价（亮点/待提升分列）
- 四层批改梯度（基础纠错绿、语言优化橙、结构逻辑蓝、文体适配紫）
- 批注预览（高亮标注 + tooltip建议）
- 操作按钮（查看优化方案、复制优化版、加入错题本、范文参考）

### 页面4：学习报告（Report）
- 周/月/年时间切换
- 六维能力雷达图（SVG绘制：发音、语法、词汇、流利度、写作、逻辑）
- 学习时长柱状图（最高值橙色高亮）
- 口语/作文评分趋势对比卡片
- 错题本入口 + 弹窗（错误列表、开始复习按钮）
- 学习成就徽章横向滚动列表
- AI数据洞察建议

### 页面5：个人中心（Profile）
- 顶部渐变头部（头像、昵称、VIP状态、到期时间）
- 数据统计（学习天数、练习次数、累计分钟）
- 能力综合评估（6维进度条 + 等级 + 超越百分比）
- 学习数据柱状图
- 错题/错音本列表
- 功能菜单（我的订单、我的收藏、学习目标、邀请好友）
- 学习目标设置弹窗（目标考试、目标分数、每日时长）
- 邀请好友弹窗（邀请码 + 复制链接）
- 设置菜单（常规设置、提醒设置、隐私政策、关于我们）
- 常规设置弹窗（深色模式、AI语音、界面语言、字体大小、清除缓存）
- 退出登录

### 页面6：全真模考（Exam）
- 考试类型选择网格（雅思、托福、四六级、高考、考研、自定义）
- 考试详情区域（名称、时长、描述、说明要点）
- 模考进行弹窗（倒计时、题目、录音按钮、转写文本）
- 模考结果弹窗（4维评分 + AI点评）
- 历史模考记录列表（从API加载真实数据）

### 页面7：专项提升（Improve）
- AI推荐今日训练卡片（基于用户弱项）
- 四大专项模块列表（发音、语法、词汇、逻辑 + 真实进度）
- 发音练习子项（音标入门、连读技巧、语调训练、口型纠正）
- 每日一练卡片（基于弱项智能推荐题目）
- 精品课程横向滚动列表
- 专项练习弹窗（题目 + 答案输入）
- 每日一练弹窗（题目 + 提示 + 答案输入）
- 练习结果弹窗（参考答案 + 积分奖励）

### 页面8：登录/注册（Login）
- 应用Logo + 标题
- 登录/注册Tab切换
- 表单（用户名、邮箱、密码 + 验证规则）
- 演示账号提示

## 后端API设计

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/auth/register | POST | 用户注册 |
| /api/auth/login | POST | 用户登录 |
| /api/auth/me | GET | 获取用户信息（含打卡记录、剩余次数） |
| /api/home/dashboard | GET | 首页仪表盘数据 |
| /api/speak/scenes | GET | 口语场景列表 |
| /api/speak/chat | POST | 口语AI对话 |
| /api/write/topics | GET | 作文题目列表 |
| /api/write/correct | POST | 作文AI批改 |
| /api/exam/types | GET | 考试类型列表 |
| /api/exam/result | POST | 提交模考结果（AI评分） |
| /api/exam/history | GET | 模考历史记录 |
| /api/report | GET | 学习报告数据 |
| /api/report/error-book | GET | 错题本数据 |
| /api/improve/modules | GET | 专项模块（真实进度） |
| /api/improve/pronunciation | GET | 发音练习子项 |
| /api/improve/courses | GET | 精品课程列表 |
| /api/improve/daily | GET | 每日一练（智能推荐） |

## 核心业务逻辑

- 学习数据实时聚合：连续天数、总时长、总次数
- AI评分对标雅思/托福官方标准（0-9分制）
- 四层批改梯度：基础纠错→语言优化→结构逻辑→文体适配
- 自适应推荐：基于用户雷达图弱项推荐训练内容
- 错题自动收录：作文批改中的错误自动写入错题本

- backend/ — 完整的FastAPI后端项目（可直接python run.py启动）
- README.md — 项目说明和启动指南
